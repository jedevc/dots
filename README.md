# dots

These are my dot files.

## Installation

Install everything (probably not what you want):

```bash
$ git clone https://github.com/jedevc/dots.git
$ cd dots
$ python -m dots install all
```

Install only zsh themes:

```bash
$ python -m dots install zsh
```
